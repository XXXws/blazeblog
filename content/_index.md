# 欢迎来到BlazeBlog

BlazeBlog是一个现代化的技术博客平台，采用JAMstack架构，提供卓越的性能和用户体验。

## 特色功能

- 🚀 高性能：Lighthouse评分>95分，LCP<2.5s
- 🌙 暗黑模式：完整的主题切换支持
- 🔍 智能搜索：基于Fuse.js的实时搜索
- 📱 响应式设计：完美适配所有设备
- ♿ 可访问性：遵循WCAG AA标准
- 💬 现代评论：基于GitHub Discussions
